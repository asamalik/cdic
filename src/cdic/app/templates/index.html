DOPR Other Package Repositories

What is it about?

Dopr is a simple service to create docker images with embedded copr repos as a software sources.

Current workflow consists of the following steps

  • Login using your fedora openid
  • Create new project
  • Provide source Dockerfile and link copr repos
  • Press and wait for results
  • Use docker pull <repo-name> like with any other public docker image
